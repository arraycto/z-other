# sell

> 仿饿了么商家点餐

## Build Setup

``` bash
# 分别给vue-music和server安装依赖
npm install

# 启动服务器代理3003端口
node server/bin/music

# 启动本地服务 localhost:8080
npm run dev

# 编译打包dist
npm run build
```
